Step 1: Which language to learn?

The world of coding is quite diverse and populated. To start, there are a number of languages to choose from...PHP, JavaScript (not to be confused with Java), Ruby, HTML, Python, and Lua not to mention the C's: C, C++, C#, and Objective-C.

Each type of language has strength and weaknesses that are determined by the needs of the project at hand. In the broadest terms there are layers of coding:

  • High Level
  • Low Level

High Level language, according to Wikipedia is a programming language with strong abstraction from the details of the computer. In other words higher levels are used to make things that are more software oriented such as video games, websites and applications for the iPhone, iPad or computer.

Low Level language, according to Wikipedia is a programming language that provides little or no abstraction from the computer's instruction set architecture and generally refers to either machine code or assembly language. Put into simpler terms, lower levels are used to make things that are closer to the hardware. This may include the software on a microprocessor controlling a robot to the chip in your keyboard used to interpret key presses.

It should be noted that no matter what language you learn first, the second will always be easier and the third even more. So don't get too hung up on what the best language is and just get started.
